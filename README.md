# 🌸 template-web

Template base para projetos web com React + TypeScript + Tailwind + React Router.

---

## Stack

- React 19
- TypeScript 5
- Vite 7
- Tailwind CSS 4
- React Router v7
- ESLint + Prettier

---

## Estrutura
src/
components/   — componentes reutilizáveis
hooks/        — hooks customizados
pages/        — páginas da aplicação
App.tsx       — rotas
main.tsx      — entrada

---

## Como usar

**1. Clone o template:**
```bash
git clone https://github.com/yannevic/template-web.git nome-do-projeto
cd nome-do-projeto
```

**2. Instale as dependências:**
```bash
npm install
```

**3. Rode em desenvolvimento:**
```bash
npm run dev
```

**4. Troque a origin pro seu repositório:**
```bash
git remote set-url origin https://github.com/seu-usuario/seu-repo.git
```

---

## Scripts

| Comando | O que faz |
|---|---|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera a build de produção |
| `npm run preview` | Visualiza a build localmente |
| `npm run format` | Formata o código com Prettier |
| `npm run lint` | Verifica erros com ESLint |
